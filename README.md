# Hangman Game

A simple text-based Hangman game built using Python.

## Features
- Random word selection
- 6 incorrect attempts
- Input validation
- Displays guessed letters
- Replay option

## Technologies Used
- Python

## Concepts Used
- random module
- while loops
- if-else conditions
- lists
- strings

## How to Run

```bash
python main.py
```

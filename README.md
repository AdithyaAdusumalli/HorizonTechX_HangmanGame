# 🎮 HorizonTechX Hangman Game

A Python-based word guessing game developed as part of the Horizon TechX Python Programming Internship Program.

The Hangman Game is a classic word-guessing challenge where players attempt to discover a hidden word one letter at a time. The player is allowed a limited number of incorrect guesses before the game ends.

This project demonstrates core Python programming concepts including loops, conditional statements, lists, strings, functions, and random word selection.

---

## 📌 Project Overview

The objective of this project is to create an interactive Hangman game using Python.

The game randomly selects a word from a predefined list and asks the player to guess the word by entering one letter at a time.

The player wins if all letters are guessed correctly before running out of attempts.

## 🏗️ Game Workflow

Start Game

↓

Random Word Selected

↓

Display Hidden Letters

↓

Player Guesses Letter

↓

Correct Guess?

├── Yes → Reveal Letter

└── No → Reduce Attempts

↓

Word Completed?

├── Yes → Player Wins

└── No → Continue

↓

Attempts Exhausted?

├── Yes → Game Over

└── No → Continue
